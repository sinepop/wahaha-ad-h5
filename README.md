1. 首页点击「开始检测」
2. 完成8道童年选择题
3. 获得专属AD人身份档案（含童心值/含钙量/灵魂浓度/伪装能力）
4. 截图分享到朋友圈

## 技术说明

- 单文件HTML，Base64内嵌全部素材（图片+音频），可离线打开
- 纯前端，无后端依赖
- Google Fonts 字体（需联网加载）
- html2canvas 实现高清截图分享
